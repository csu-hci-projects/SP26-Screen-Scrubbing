## How To Run

- Unity 6 

- XR Interaction Toolkit installed
  
- Start with controllers disabled in HandTracking mode


## Controls

- **Hand Tracking** — hover and pinch to select clip to move, or to select handles to resize. Move hand to move clip or resize accordingly.

- **Controllers** — hover and use side trigger to select clip to move, or to select handles to resize. Move controller to move clip or resize accordingly.
